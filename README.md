## 🧠 Overview

SwishFormer is an AI-powered framework that uses a wearable thumb-mounted DIGIT sensor to estimate fruit firmness and ripeness from visual tactile imagery. The proposed model processes tactile videos of fruit palpation to produce real-time predictions, offering a non-destructive alternative to traditional methods.
